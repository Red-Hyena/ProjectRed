Who I am

I am Anthony Dixon, a web developer from Blyth, Northumberland.

I have always been interested in IT and how it works. From playing width Blender animations, Unreal Engine and Linux, I'm always looking to learn new skills.

I found a passion for web design, with a focus on mobile responsive and accessible designs and an interest in using SVG's with CSS animations.

The design

I always start with Figma as it allows me to quickly get ideas down. I also use Figma to plan for mobile responsiveness and accessibility.

For images I use Gimp, Krita, Inkscape and if I'm feeling fancy Blender.

The development

For development I prefer pure HTML, CSS and JavaScript with a little bit of SASS. I use Git for project management.

I have just started learning jQuery for including libraries, although I prefer to tailor make each script with JavaScript for each project. This allows me to implement tidy and efficient scripts.
